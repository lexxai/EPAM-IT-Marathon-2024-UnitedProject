# Project Setup

## Steps to Run

1. **Copy `.env_example` to `.env`**

   ```bash
   cp .env_example .env
   ```

2. **Fill in the required information inside `.env`**

   (For example: passwords, database names, API keys, etc.)

3. **Build and start the Docker containers**

   ```bash
   docker compose up --build
   ```
